# Creative Canvas - Drawing App

An interactive and aesthetic browser-based drawing application built with HTML5 Canvas, CSS, and JavaScript. Designed with a sleek sidebar UI, this app allows users to freely draw, erase, undo/redo strokes, and save their artwork as a PNG.
